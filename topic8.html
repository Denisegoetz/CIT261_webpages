Topic: Manipulating CSS Class Properties Using JavaScript

Manipulating CSS Class Properties

In the previous topics, DOM Manipulation, we learned how to manipulate the HTML of a page, but it is also possible to manipulate the CSS of the page. Style properties can be changed by changing the style attribute inside the element in the HTML, and this will override any setting in the CSS style sheet. Throughout the next section there are examples of adding and changing CSS attributes.

The syntax for for changing a CSS style is shown below. Notice that the style property is written in camelCase rather than hyphenated. Where a CSS style property would normally have a hypen, for example, font-size:, it would be fontSize instead.

element.style.styleProperty = "value";

This works, but it is not the best solution. It adds an inline CSS style to the page instead of keeping the CSS separate from the HTML. A better approach is to add a class name to the element that refers to a styling rule in the CSS document for the page.

element.classList.add('className');

Altering one Element and one CSS property

//Change one property with style rather than
//using cssText and taking a chance with a typing error - high probability
document.querySelector("h3").style.color = "green";

Altering one Element and multiple CSS properties

//You write a string of CSS as if in the style sheet itself

document.querySelector("h4").style.cssText = "color: blue; font-size: 35px;";
