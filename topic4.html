Topic: JSON Parse, Stringify

JSON

JSON is a JavaScript data-storage format that can be used to exchange information between web services. In JSON notation, an object is formatted as a string. When written in JSON, property keys need to be in double quotes. Values can include strings in double quotes, numbers, booleans, null, arrays and objects. Methods are not allowed.

JSON Parse

The JSON parse method will take a JSON string and turn it into a JavaScript Object, which is the opposite of stringify.

This first example is doing WAY MORE than needed to provide an example of JSON Parse. Here, I have made a call to a Star Wars API website that gives me a JSON string, JSON.parse puts that data into an object. Then I access a few fields in that object to produce the names of StarWars ships and classifications.

var container = document.getElementById("jsonGet");

function jsonGetDemo() {
var myRequest = new XMLHttpRequest();
myRequest.open('GET', 'https://swapi.co/api/starships/');
myRequest.onload = function() {
var myData = JSON.parse(myRequest.responseText);
var htmlString = "";
for (i=0; i < myData.results.length; i++){
htmlString += "

" + myData.results[i].name + " -  " + myData.results[i].starship_class + " ship classifcation

";
}
container.insertAdjacentHTML("beforeend", htmlString);
};
myRequest.send();
}

Star Wars Ships - API Website: https://swapi.co/

JSON Stringify

The JSON stringify method will take a JavaScript object and turn it into a JSON string. Notice that the string doesn't have the message method in it. Functions are not allowed, so JSON.stringify skips over it.

// Create Object
let actor = {
fname: "Chris",
Lname: "Hemsworth",
actor: true,
age: 35,
features: ["Tall","Blond","Buff","Australian","Married"],
message: function() {return "You are my hero!";}
};

// stringify object Avenger = JSON.stringify(actor);
