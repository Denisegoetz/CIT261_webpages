Objects

In JavaScript, objects provide a way for us to store, manipulate, and send data over the network. Object are containers for variables. The names of these variables (for ex. firstName) are called keys, and are used to access the value inside the variable. The variables in an object are officially called properties. Think of objects as a collection of ‘key:value’ pairs. Here is a simple object.

var person = {
firstName: ‘John’,
lastName: ‘Smith’
};

Properties

A property of an object can be explained as a variable that is attached to the object. Object properties are basically the same as ordinary JavaScript variables, except for the attachment to objects. The properties of an object define the characteristics of the object.

Creating Objects

There are several ways to create objects. I have demonstrated 2 ways. The object person example was created using the Object constructor, which is using the ‘new’ keyword with the in-built Object constructor function. The object customer example used object initialization.

Keyword New and Object Constructor

You can create an object using the ‘new’ keyword, in this case, you need to have a constructor function which looks like this. This method is longer to code and not recommended because scope-resolution needs to occur (behind the scene) to determine if constructor function is built-in or user-defined. Unassigned properties of an object are undefined (and not null).

var person = new Object(); //create object with new keyword.

person.firstName = ‘John’; //now add properties to person object
person.lastName = ‘Smith’; //lastname is a property of person and key to get the value

Object Literal Syntax

Another way to create an object is declare and initialize the object in the same step. Simply declare a new variable and assign a comma separated list of key-value pairs inside curly brackets.

var customer = {
firstName: ‘Sue’,
lastName: ‘Brown’,
customerId: 4416
};
