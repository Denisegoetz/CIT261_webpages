Topic: Designing, Defining, and Triggering CSS3 Transitions without Custom Libraries

CSS Transitions

We remember without transitions, changes to CSS take effect immediately, which can be a jarring user experience. Transitions allow you to make those changes gradually, slowly changing until it's final state.

Basic Accordion - Made from CSS only

This accordion is created with CSS only. The transition properties of CSS make the displaying of the text smooth. That is done with transitioning the height from 0 to whatever height you want to view it at. Notice how on a hover the color is trasitioned to a lighter one to let the user know what they are about to select. One drawback is that the accordion leaves a section open all the time. Also, in this example, you can only open one section at a time.

Accordion - even better with JavaScript

JavaScript Accordion

This accordion is created with CSS and JavaScript. The transition properties of CSS make the displaying of the text smooth. This is my favorite accordion because it neatly collapses and if you want to have all the sections opened at once you can. As a user, it would annoy me that I have to memorize or refer back to the previous section when only one section can be opened at a time.
