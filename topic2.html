Topic: HTML5 Tags - Video, Audio, and Canvas

Audio Tag

The HTML audio tag plays embedded audio files and the supported file formats are mp3, wav, and ogg. The opening audio tag can have attributes similar to the video tag, such as controls, autoplay, loop, muted, volume, etc. It allows for one or more source tags for flexiblity. For example, if the first mp3 isn't supported by the users browser, it will try the ogg file and keeps going down the list until something works or the message is displayed.

<audio controls>
  <source src="audio.mp3" type="audio/mp3">
  <source src="video.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

Video Tag

The video tag makes it easy to embed videos into a website. HTML5 video includes an opening and closing video tag, with one or more source tags to provide alternate video file formats.

<video width="300" height="200" controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

Supported video formats include mp4, webm, and ogg. Providing multiple formats make it more likely to have a video format that is supported by the user's browser. The first recognized format is used.

The opening tag can have several attributes.

  1. Providing height/width attributes keep the webpage from flickering as the video loads.
  2. controls will cause controls like pause, play, etc to show in the video window.
  3. autoplay will automatically play the movie on load. This is usually not a good idea as it may annoy users.
  4. loop will make the video repeat automatically.
  5. poster="image.jpg" will display the image in the link until the video plays.
  6. An optional src attribute can be used in the opening tag to provide the video url instead of in a <source> tag.
  7. There are many other attributes that can be added to a video tag.

YouTube Videos

To include a YouTube video on your webpage, you need to embed it with an iframe. You can copy the Embeded link by selecting SHARE in YouTube and clicking on Embed. There are some options you can deselect or select. Then you copy the code and insert it into the HTML. There is one problem with YouTube iframes. To get the screen to show up at a decent size and proper ratio you need to "trick" the iframe using some CSS. Enclose your iframe in div with a class and use the CSS below. 56.26% is supposed to be a magical ratio for the screen size.

/* This is to fake out to get a full sized video on your page */
/* .youtube=container and .youtube iframe= makes video as large as container */

  .youtube{
   position: relative;
   padding-bottom: 56.25%;
   padding-top: 25px;
   height: 0px;
}
  .youtube iframe{
   position: absolute;
   left: 0;
   right: 0;
   top: 0;
   bottom: 0;
   height: 100%;
   width: 100%;
}

Canvas tag

The canvas tag is a container for graphics, which can be as large as the entire screen or much smaller. Javascript can be used with the canvas tag to enable drawing on a webpage, displaying graphics, animating objects, etc.

Define tag in HTML

<canvas id="myCanvas" width="300" height="300">
  Your browser does not support the canvas tag.
</canvas>

The id is necessary to access the canvas through JavaScript. Width and height can also be used to define the size of the canvas. Once the canvas is defined in the HTML, use JavaScript to draw lines, shapes, and text, display and animate images, etc. You should also add a text default for browsers that don't support the canvas tag.

Draw with JavaScript

In order to use the canvas, JavaScript is needed. First, select the canvas element using getElementById().

Next define a rendering context using getContext(2d). (Other contexts, such as 3d are available.)

Then define what you want to draw in the canvas element. Define the fill color or pattern with fillStyle, make a rectangle with fillRect(), a circle with arc(), text with fillText(), or draw a line with lineTo(). Finally, to draw the shape or line add stroke(), and to fill the shape add fill(). The code below is for a rectangle.
